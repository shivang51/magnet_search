## Magnet Search
A simple cli to search for magnet links and directly open them in default torrent client.

### Usage
```bash
# Search for a movie
magnet-search "Inception"
# Search for a TV show
magnet-search "Breaking Bad"
```

### Installation
```bash
# After cloning the repo, run the following
cargo install --path .
```
